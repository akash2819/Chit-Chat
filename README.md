
# Chit Chat 

It is a chat room created in C. it can hadle upto 10 clients.

## Run Locally

Clone the project

```bash
  git clone https://github.com/akash2819/Chit-Chat.git
```

Go to the project directory

```bash
  cd  Chit-Chat
```

Start the server

```bash
./server
```
Open another terminal 
```bash
./client
```
